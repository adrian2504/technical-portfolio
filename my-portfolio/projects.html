üì∞ Intelligent Content Aggregator

Serverless content aggregation platform that combines RSS feeds, GitHub activity, and social media into personalized daily digests. Uses Amazon Bedrock to generate concise summaries of each URL, making it easier to stay current with technology trends.

Problem Solved

Staying current with technology trends requires monitoring multiple sources. Manual aggregation is time-consuming and reading every article is overwhelming.

Technical Approach

Built serverless architecture with Lambda functions for content collection, Amazon Bedrock for AI-powered summarization, and SNS for notifications. Each URL gets a concise summary to help users quickly identify relevant content.

Current Status

Active development - adding SNS subscription management, expanding RSS search terms, and refining Bedrock prompts for better summaries.
