# Documents Folder

This folder contains downloadable documents for the portfolio.

## Files to Add:

1. **Adrian-Profile.pdf** - Your main profile document
   - This should be your most up-to-date resume/profile in PDF format
   - The filename should match exactly: `Adrian-Profile.pdf`

## How to Add Your Profile:

1. Save your resume/profile as a PDF file
2. Name it exactly: `Adrian-Profile.pdf`
3. Place it in this `static/documents/` folder
4. The download functionality is already set up in all HTML files

## Current Download Path:
The JavaScript function looks for: `static/documents/Adrian-Profile.pdf`

If you want to use a different filename, update the `profileUrl` variable in the JavaScript functions across all HTML files.
